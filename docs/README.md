### 🙋 입력

- [X] 월과 시작 요일(일, 월, 화, 수, 목, 금, 토)
    - [X] 빈값인지
    - [X] split 했을 때 각각이 빈값인지 확인
    - [X] 월은 숫자인지 IntOrNull
    - [X] 월은 1~12 사이 숫자인지
    - [X] 요일은 일, 월, 화, 수, 목, 금, 토 중에 값이 있는 지 확인

- [X] 평일 비상 근무 순번, 휴일 비상 근무 순번
    - [X] 빈 값인지 확인
    - [X] 각 닉네임이 빈값인지 확인
    - [X] 중복 안됨 : 수아가 두 번 편성된 경우, 닉네임이 동일한 경우
    - [X] 닉네임은 최대 5자
    - [X] 사람 수는 최소 5명에서 35명



### 🖥 출력
